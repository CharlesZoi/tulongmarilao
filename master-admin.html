Dashboard Help & Guide

Pins Table

View and manage all reported relief locations:

  • Search: Find locations by name, source, or reporter
  • Filter by Urgency: Show Critical, Urgent, or Moderate reports
  • Filter by Status: View reached or not-reached locations
  • Sort: Order by date, urgency, or name
  • Actions: Use the 3-dot menu for View, Open map, Edit, and Delete

Supported Locations

Manage support progress and confirm arrivals:

  • Search: Find active support by location, responder, or source
  • Support Status: Filter on-the-way, proof-submitted, or reached locations
  • Actions: Primary button changes by state: Ping for proof, Review proof, or Undo reached
  • More Actions: 3-dot menu for Donation workflow, View, Edit, Open map, or Delete

Donation Logs

Review and verify submitted donation deliveries:

  • Search: Find donations by donor name or location
  • Delivery Status: Filter by on-the-way, proof-submitted, or reached donations
  • Actions: View details, approve/reject pending logs, and undo decisions

Master Admin Features

Special capabilities available to master administrators:

  • Delete ANY pin: Full deletion privileges for all locations
  • Admin Map: Access admin map with enhanced capabilities
  • Import/Export: Download Excel reports and import location data
  • Real-time updates: Live data synchronization

Tips & Best Practices

  • Use the Refresh button to sync latest data
  • Download Excel reports for offline analysis
  • Verify donation deliveries promptly
  • Mark locations as reached when relief is delivered
  • Use the Admin Map for geographical view
